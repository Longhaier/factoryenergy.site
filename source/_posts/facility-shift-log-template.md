# 厂务值班日志模板分享

## 值班日志有多重要？

讲个故事。

2018年某个周末晚上，厂里空调主机报故障，值班师傅简单写了个"空调故障"就下班了。周一上班，维修班去现场一看，压缩机已经烧毁——如果当晚及时处理，最多换个电磁阀就能解决的问题，最后花了8万块换压缩机。

**值班日志不是流水账，是保命符。**

## 我用过的日志模板

分享一个我们在用的模板，好用不复杂：

```
【厂务值班日志】
日期：2026-03-17
值班人：张三
班次：白班/夜班

【设备运行情况】
1. 空调系统：正常 / 异常（详情：____）
2. 电力系统：正常 / 异常（详情：____）
3. 给排水：正常 / 异常（详情：____）
4. 消防系统：正常 / 异常（详情：____）

【巡检记录】
- 08:00 配电室巡检：温度____湿度____
- 10:00 空调机房巡检：主机运行____号
- 14:00 水泵房巡检：出水压力____MPa
- 16:00 末端巡检：各区域温度____℃

【异常记录】
- 时间：____
- 设备：____
- 现象：____
- 处理：____
- 结果：____

【交接事项】
1. ____
2. ____

【备注】
____
```

## 关键点：怎么写好异常记录？

这是我最想强调的。很多值班日志写着"设备异常"，然后就没有然后了。

**合格的异常记录要包含：**
1. **时间** - 什么时候发现的
2. **现象** - 具体什么情况（报警代码、参数异常、异响等）
3. **初步判断** - 可能是什么原因
4. **处理措施** - 做了什么
5. **后续建议** - 需要谁来跟进

## 电子化推荐

现在很多厂都在用电子化的值班系统了。推荐几个我们试过的：

- **钉钉/飞书日志** - 手机就能填，支持模板
- **SCADA系统** - 直接对接设备，自动记录参数
- **专门厂务软件** - 设施狗、掌厅等

电子化的好处是：可以追溯、可以统计、不会丢。

## 写在最后

值班日志这个东西，要么不写，要写就认真写。

不是为了应付检查，是为了**关键时刻能救场**。
